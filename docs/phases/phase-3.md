# PHASE 3: FIRST PRODUCTS (2030–2032)
### "One protocol. One niche. One paying customer."

## OBJECTIVE
Stop prototyping. Start shipping. Generate first revenue.

## WHAT TO EXPECT
- The gap between prototype and product is enormous. Expect 3x the time you planned.
- Your first customer will break everything. That is their job.
- By end of phase: at least one product generating recurring revenue.
- ArcVault may generate revenue before any tech product does.

## BUILD PRIORITY ORDER

### Build 1: ContextCore — Hollywood Production Accounting Vertical
**Why first:** You know this world. You have data access. Lowest cold-start risk.

Steps:
- [ ] Interview 10 production accountants. Document every pain point.
- [ ] Build document ingestion pipeline (Rust + Python)
- [ ] Fine-tune model on production accounting corpus
- [ ] Build simple chat interface on top
- [ ] Find 3 beta production companies (offer free for 6 months)
- [ ] Charge at month 7

**Expected revenue:** $500–$2,000/month per company at beta. $5,000–$15,000/month at scale per vertical.

**Resources:**
- LlamaIndex documentation (data ingestion for LLM apps)
- Unstructured.io (document parsing library)
- Modal.com (GPU infrastructure for fine-tuning)
- *The Mom Test* — Rob Fitzpatrick (how to interview customers without lying to yourself)

### Build 2: Canonchain MVP
**Why:** Establishes IP infrastructure for everything else. Simple to build relative to ZK products.

Steps:
- [ ] Deploy rights registry on StarkNet mainnet
- [ ] Build simple web interface for registration
- [ ] Register every film, script, score you've ever made
- [ ] Approach 5 independent filmmakers to test
- [ ] Document legal standing in 3 jurisdictions

**Expected revenue:** $50–$200 per registration. Volume business.

### Build 3: ArcVault First Acquisition
**Why:** Revenue without deep tech infrastructure. Validates the IP arbitrage thesis.

Steps:
- [ ] Build signal detection script (scrape bestseller lists in PT, ES, KO)
- [ ] Identify first acquisition target
- [ ] Negotiate rev-share rights (target: 50/50 new market revenue)

**Expected revenue:** Variable. First acquisition: $10K–$100K over 18 months if story is well-chosen.
